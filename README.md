# marvel-client
Conector REST (Marvel API) utilizando Java 7, Maven, Spring MVC, Rest, Spring Cache, Json, EclEmma, CPD, Junit, Sonar.

# Getting Started

clone from
[GIT](https://github.com/diegomarinho/marvel-client) and then
use Maven (3.0.\*) and Java (1.7 or better):

    $ git clone ...
    $ mvn install -P marvel-client
